# 字符串编辑距离

## 题目描述

给定一个源串和目标串，能够对源串进行如下操作：

- 在给定位置上插入一个字符
- 替换任意字符
- 删除任意字符

写一个程序，返回最小操作数，使得对源串进行这些操作后等于目标串。

__case__

> input: horse ros <br/>
> output: 3

## 解体思路

动态规划，状态转移方程:

1) s[i] == t[j] => dp[i][j] = dp[i-1][j-1]
2) s[i] != t[j] => dp[i][j] = Math.min(dp[i-1][j-1], Math.min(dp[i][j-1], dp[i-1][j])) + 1;
